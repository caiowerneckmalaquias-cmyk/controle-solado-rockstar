# Controle de Solado - Calçados Rock Star

Projeto pronto para publicar na Vercel.

## Passos
1. Baixe este ZIP
2. Extraia a pasta
3. Suba os arquivos para um repositório no GitHub
4. Na Vercel, clique em Add New > Project
5. Escolha o repositório
6. Framework: Vite
7. Deploy

## Instalar no celular
Depois de publicado, abra o link no celular e use 'Adicionar à tela de início'.
